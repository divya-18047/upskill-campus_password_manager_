# 🔐 Password Manager

## 📌 Project Description
This is a Password Manager application developed using Python and SQLite. It allows users to securely manage website credentials.

## 🚀 Features
- Add Password
- View Saved Passwords
- Search Password
- Delete Password
- Generate Strong Password

## 🛠 Technologies Used
- Python
- SQLite3
- VS Code

## 📂 Project Files
- database.py
- main.py
- view_passwords.py
- search_password.py
- delete_password.py
- password_generator.py

## ▶️ How to Run

1. Run `database.py` to create the database.
2. Run `main.py` to add passwords.
3. Run `view_passwords.py` to view saved passwords.
4. Run `search_password.py` to search passwords.
5. Run `delete_password.py` to delete passwords.
6. Run `password_generator.py` to generate strong passwords.
